## AudiveX – Headphone Product Showcase Website

AudiveX is a modern **product showcase landing page** designed for headphone products.
The project focuses on creating a clean UI and interactive user experience using **HTML, CSS, and JavaScript**.

## Features

* Modern hero section with product highlight
* Smooth horizontal **product slider**
* Interactive UI animations and hover effects
* Floating product animation for better visual experience
* Contact section with **email and call actions**
* Responsive and clean layout design

## Technologies Used

* HTML5
* CSS3 (Flexbox, animations, gradients)
* JavaScript (DOM manipulation, slider functionality)

## Purpose of the Project

This project was built to practice **frontend development, UI design, and interactive JavaScript features** while creating a real-world product landing page.

## Project Preview

A stylish landing page that showcases headphone products with smooth animations and a modern interface.

## Future Improvements

* Make the website fully responsive for mobile devices
* Add product filtering and search functionality
* Integrate a real shopping cart system
